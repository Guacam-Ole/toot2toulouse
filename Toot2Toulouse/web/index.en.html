What is this all about?

Toot2Toulouse(t2t) is a crossposter that publishes your toots(Mastodon) as tweets(Twitter). Before you decide if you want to use it let's make sure you know what it does (and what it doesn't). Please check if this fits your needs before signing up

What it does::
  • Publish your public toots as tweets in Twitter
  • Add Content-Warning Information on the tweets
  • Add the following Media types: GIF, JPG, WEBX, MP4 (please look at the server section for file limits)
  • Split longer toots into tweet-threads
  • Delay tweeting your toots

What it doesn't:
  • remain the "sensitive" flag (e.g. from content warnings)
  • Crosspost into the other direction. So NO Twitter -> Mastodon crossposts
  • Allow any other than public posts on Twitter
  • Allow any other media type
  • Tweet your replies to other toots
  • Tweet boosts

What it can't do yet: (working on that)
  • Replace mentioned users on Mastodon by the according Twitter-Account
  • Tweet your replies to your own toots

Server Limits and Settings

Because Twitter has limitations when using the (free) API this server might have some limitation what you can do and what you can't do. This can not be changed. If you need more resources try to find another t2t-instance or Host your own
